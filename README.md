  # 💐📖 emory 📖💐
  
  絵文字だけで日記を書いて、書いた日記を友達とシェアするサービスです。
  
  
  ![twitter_share](https://user-images.githubusercontent.com/76583053/134182713-fb2ff03f-6ce9-42f4-8ed0-c585771b6f8e.png) 
  
  <br>
  
  ## 🌼サービス概要
  🦖絵文字”だけ”で日記を書くので、  
  簡単にそして直感的に感情を表すことができます。   
  🐊そしてグラフとカレンダーの色をみて  
  日々の感情の変化を知ることができます。   
  🦕また日記を所属コミュニティ内でシェアできるので、  
  コミュニケーションのきっかけ作りになるサービスです。  
  
  <br>
  
  ## 🌼emoryを勧めたいユーザー
  🐶日記をつけたい気持ちはあるけれど  
  忙しい日々の中でなかなか続けて書くことが難しい人  
  🐻毎日顔を合わせることが難しくなった現在、  
  所属コミュニティ内でのコミュニケーションのきっかけを作りたいと思っている人  

  <br>
  
  ## 🌼emoryで解決できること
  🍉絵文字だけで直感的に日記をつけられて、  
  またLINEを使ってリマインドをすることで日記のつけ忘れを防ぎます。   
  🍋所属コミュニティ内で絵文字日記をシェアすることで、  
  感情や日々の出来事などがわかって、会話のきっかけ作りになります。    
  🍊悩みを抱えて落ち込んでいる人に気付きやすいように  
  感情の変化を表すグラフを用意しています。  

  <br>
